#  Cyber Log Analyzer

A GUI-based cybersecurity tool that analyzes log files and detects suspicious activities such as brute-force attacks and abnormal traffic patterns.

---

##  Features

-  Analyze real log files (SSH, Web logs)
-  Detect failed login attempts
-  Detect successful logins
-  Analyze HTTP requests and status codes
-  Identify top active IP addresses
-  Detect suspicious activity
-  Detect potential attackers
-  Visualize data using charts

---

## 🛠 Technologies Used

- Python
- Tkinter (GUI)
- Matplotlib (Charts)
- Regex (Log parsing)
